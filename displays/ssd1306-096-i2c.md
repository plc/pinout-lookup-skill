# SSD1306 0.96" OLED I2C

Source: https://cdn-shop.adafruit.com/datasheets/SSD1306.pdf

128x64 pixels, I2C interface, SSD1306 driver. 3.3V-5V compatible.

## Pinout (display facing you, pins at top)

```
  +---------------------+
  | GND  VCC  SCL  SDA  |
  |  o    o    o    o   |
  |                     |
  |                     |
  |    0.96 inch OLED   |
  |                     |
  |                     |
  | (O)             (O) |
  +---------------------+
```

## Pin Description

- GND -- Ground
- VCC -- Power, 3.3V or 5V
- SCL -- I2C clock
- SDA -- I2C data

## I2C Address

Default: 0x3C. Some modules use 0x3D. Check/change via solder jumper on back if present.

## Wiring to XIAO ESP32-S3

SSD1306: pins 1-4 left to right, display facing you.
XIAO: L1-L7 left, R1-R7 right, top to bottom, USB at top.

| SSD1306 | #  | XIAO ESP32-S3 | #  |
|---------|----|---------------|----|
| GND     | 1  | GND           | R2 |
| VCC     | 2  | 3V3           | R3 |
| SCL     | 3  | D5 (GPIO6)    | L6 |
| SDA     | 4  | D4 (GPIO5)    | L5 |

## Wiring to XIAO nRF52840

SSD1306: pins 1-4 left to right, display facing you.
XIAO: L1-L7 left, R1-R7 right, top to bottom, USB at top.

| SSD1306 | #  | XIAO nRF52840 | #  |
|---------|----|---------------|----|
| GND     | 1  | GND           | R2 |
| VCC     | 2  | 3V3           | R3 |
| SCL     | 3  | D5 (P0.05)   | L6 |
| SDA     | 4  | D4 (P0.04)   | L5 |
